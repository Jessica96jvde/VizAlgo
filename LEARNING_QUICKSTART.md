## 🚀 How to Use

### For Stack Learning
1. Open the Learn section
2. Select "Stack" from the data structure list
3. Navigate through tabs:
   - Start with Overview for context
   - Move to Operations to learn basic functions
   - Study Algorithm tab for implementation details
   - Review Complexity for performance analysis
   - Explore Applications for real-world use
   - Follow Examples for practical understanding

### For Queue Learning
1. Open the Learn section
2. Select "Queue" from the data structure list
3. Follow the same tab progression as Stack
4. Note: Queue has 6 tabs (no Rules tab)

## ❓ FAQ

**Q: Why 7 tabs for Stack and 6 for Queue?**
A: Stack has an additional "Rules" tab because it covers specific conditions like Stack Overflow/Underflow that are more critical to understand.

**Q: Are the algorithms Kotlin-specific?**
A: No, the algorithms are provided in pseudo code format for universal understanding. They can be implemented in any programming language.

**Q: Can I use this for interview preparation?**
A: Absolutely! This covers all fundamental concepts typically asked in technical interviews.

**Q: What's the difference between Overview and Examples tabs?**
A: Overview provides definitions and general understanding, while Examples tab provides step-by-step visual walkthrough of operations.
